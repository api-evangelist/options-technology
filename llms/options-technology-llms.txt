# Options Technology

> Options Technology (Options IT) is a capital markets managed-services and trading-infrastructure provider (founded 1993, Abry Partners-backed, 550+ client firms). Its market data business — built on the 2021 ACTIV Financial acquisition — sells low-latency raw and normalized exchange feeds (colocated multicast/TCP), real-time streaming into AWS/GCP/Azure, tickerplant snapshot and time-series services, and lossless .pcap data capture. Most of the business is sales-gated and entitlement-managed; the genuinely public developer surface is the ACTIV Web One API (AtlasFeed), a WebSocket + WebAssembly JavaScript/TypeScript API with public docs, tutorials, and an in-browser playground.

## APIs

- [ACTIV Web One API (AtlasFeed)](https://weboneapi.activfinancial.com/): WebSocket (wss) market data API for browser and Node.js — snapshots, real-time subscriptions, queries, time-series history over ACTIV's normalized multi-asset feed. npm: `@activfinancial/one-api` (1.1.4). Auth is username/password inside `activOneApi.connect({ host, user, password })`; credentials are sales-provisioned and exchange-entitlement-managed (no self-serve). No OpenAPI/AsyncAPI is published.
